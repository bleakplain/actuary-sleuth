### 项目描述模板

```markdown
## 项目名称: Actuary Sleuth - AI精算审核助手

### 项目描述
基于 RAG (检索增强生成) 技术的保险产品合规性自动化审核系统，
实现法规知识检索、条款审核、报告生成全流程自动化。

Python 3.12, LlamaIndex, GLM-4, LanceDB, SQLite, 飞书API

1. 设计并实现基于 LlamaIndex + LanceDB 的 RAG 架构

4. 优化系统性能，通过连接池、缓存、批量操作提升 5 倍

### 技术成果
- 检索精度提升 20%，API 调用成本降低 30%
- 支持数千条法规实时检索，响应时间 < 500ms
- 实现批量导入性能提升 5-10 倍
```

## 🔧 重新导出

如果需要修改图表或重新导出：

### 方法 1: 使用 Mermaid Live Editor
1. 访问 https://mermaid.live/
2. 打开对应的 `.mmd` 文件
3. 编辑并预览
4. 导出为 PNG/SVG

### 方法 2: 使用 Kroki.io
```python
python3 convert_mermaid_kroki.py
```

### 方法 3: 使用 mmdc (需要依赖库)
```bash
mmdc -i input.mmd -o output.png
```
