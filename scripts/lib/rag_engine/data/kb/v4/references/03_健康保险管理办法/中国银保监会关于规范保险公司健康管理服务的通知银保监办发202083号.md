# 中国银保监会《关于规范保险公司健康管理服务的通知》（银保监办发〔2020〕83号）

## 第1项

> **元数据**: 险种大类=保险公司提供的健康管理服务，是指对客户健康进行监测、分析和评估，对健康危险因素进行干预，控制疾病发生、发展，保持健康状态的行为，包括健康体检、健康咨询、健康促进、疾病预防、慢病管理、就医服务、康复护理等。 | 险种类型=健康保险

保险公司提供的健康管理服务，是指对客户健康进行监测、分析和评估，对健康危险因素进行干预，控制疾病发生、发展，保持健康状态的行为，包括健康体检、健康咨询、健康促进、疾病预防、慢病管理、就医服务、康复护理等。

## 第2项

> **元数据**: 险种大类=保险公司提供的健康管理服务包含在保险产品责任条款中的，其分摊成本不得超过净保险费的20%，并应在条款中明确健康管理服务的主要内容，同时在精算报告中说明其定价依据。
保险公司单独提供健康管理服务的，应签订健康管理服务合同并在合同中注明服务内容和服务价格。
单独提供健康管理服务的，不得与保险产品捆绑强制销售，不得强制要求购买保险产品的客户购买健康管理服务。 | 险种类型=健康保险

保险公司提供的健康管理服务包含在保险产品责任条款中的，其分摊成本不得超过净保险费的20%，并应在条款中明确健康管理服务的主要内容，同时在精算报告中说明其定价依据。
保险公司单独提供健康管理服务的，应签订健康管理服务合同并在合同中注明服务内容和服务价格。
