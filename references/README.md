## 🔍 快速查询指南

### 按场景查询

**新产品条款审核**:
1. 先查看 `02_负面清单.md` 了解22个违规点
2. 根据产品类型查看对应产品开发规范
3. 必要时查阅 `01_保险法相关监管规定.md` 确认法律依据

**法规查询**:
- 条款编号查询 → `01_保险法相关监管规定.md`
- 费用率规定 → `03_条款费率管理办法.md`
- 信息披露要求 → `04_信息披露规则.md`

**定价合理性判断**:
- 行业标准对比 → `03_条款费率管理办法.md`
- 产品类型规定 → 对应产品开发规范
